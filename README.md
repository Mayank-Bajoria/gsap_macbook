# Apple 3D Product Viewer

A stunning, interactive 3D product showcase built with React, Three.js, and GSAP. This application features smooth animations and interactive 3D models, inspired by Apple's product landing pages.

## ✨ Project Features

- **Interactive Product Viewer:** Explore the MacBook Pro in full 3D with intuitive orbit controls.
- **Dynamic Customization:** Switch between 14" and 16" size models and instantly toggle between Space Gray and Dark color finishes.
- **Scroll-Linked Storytelling:** Experience a cinematic scroll journey where the 3D MacBook rotates synchronously as you scroll.
- **Synchronized Video Textures:** The MacBook's screen dynamically plays pre-loaded feature videos perfectly timed with scroll milestones.
- **Responsive 3D Scaling:** The 3D models automatically scale and position themselves beautifully across mobile, tablet, and desktop displays.

## 🛠️ Tech Stack

- **Framework:** [React 19](https://react.dev/)
- **3D Rendering:** [Three.js](https://threejs.org/), [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber/), [@react-three/drei](https://github.com/pmndrs/drei)
- **Animations:** [GSAP](https://gsap.com/) & [@gsap/react](https://www.npmjs.com/package/@gsap/react)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **State Management:** [Zustand](https://github.com/pmndrs/zustand)
- **Build Tool:** [Vite](https://vitejs.dev/)
